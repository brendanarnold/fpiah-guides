Windows Phone Development for People in a Hurry

Introduction

These guides,

  • are for people who know a bit about programming but have not programmed much in a Windows Phone environment
  • are written for people with the limited capabilities of the free developer tools
  • makes many decisions for you based on developer practice. If you do not like a particular practice, feel free to adapt to your needs
  • are written to be only slightly more in depth than a cheat-sheet to enable fast reference. Microsoft already has massively detailed documentation available
  • cover 2D Silverlight apps. For writing 3D apps, see XNA development on Windows Phone
