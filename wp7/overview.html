Windows Phone Development for People in a Hurry

Overview

Architecture

  • The Windows Phone platform is a subset of Silverlight which is itself a subset of the Windows Presentation Foundation. When searching for solutions first try including 'Windows Phone' then try 'Silverlight', then try 'WPF'
  • The UI for all of the above is written in 'eXtensible Application Markup Language' (XAML) and the backend is written in either C# or Visual Basic (VB)
  • Windows Phone 8 may also be written in Native Code (C++) and HTML/Javascript
  • XAML/C# is directly comparable to HTML/Javascript however,
    • XAML/C# are much cleaner languages
    • XAML/C# are tailored for the task of writing applications
    • HTML/Javascript apps takes less changes to port to another platform
  • .NET is,
    • a huge class library
    • the virtual machine that runs compiled C# or VB code
    • the name for the platform as a whole

Visual Studio Express

  • Visual Studio is a development environment optimised for writing C#/VB code
  • There is some support for designing XAML pages in Visual Studio but the experience is better in Blend
  • Editing XAML code directly is a better experience in Visual Studio
  • The main difference between the free Express edition of Visual Studio and the full version is that you cannot install plugins in the Express edition

Expression Blend

  • Expression Blend is a design tool that is optimised for generating XAML
  • You can write C#/VB in Blend but the experience is not as good as in Visual Studio
