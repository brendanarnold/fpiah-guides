Expression Blend

By far the best way to learn about using Expression Blend is to watch a video of someone using it. A great introduction is Corinna Black's talk 'How to: Building a Prototype App Using Expression Blend for Windows Phone' with the following notes,

  • I would recommend creating data as she does but only creating one data set and re-create your View-Model object
  • Although there is a 'Show Snap Grid' button at the bottom of the design window, the grid mentioned is part of a deeper design philosophy. This video at 6:48 explains.
