Interface
A template for a class that specifies properties and methods to implement. (e.g. 'IValueConverter' interface requires the methods 'Convert' and 'ConvertBack' to be implemented.) Rather than writing code that only accepts a particular class, code can instead be written that accepts any object so long as it implements a particular interface. Generally .NET interfaces begin with the letter 'I'
